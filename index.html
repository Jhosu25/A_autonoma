Document Object Model (DOM)

Contenido sacado del libro: Programación de Páginas Web con JavaScript y PHP (Mercedes Escarcena, 2017)

¿Qué es el DOM?

El DOM (Modelo de Objetos del Documento) es una interfaz de programación para documentos HTML y XML. Representa la estructura del documento como una jerarquía de nodos y objetos que permite acceder y manipular el contenido, la estructura y el estilo de la página.

Características principales

  • Representa el documento como una estructura en forma de árbol.
  • Cada nodo del árbol representa una parte del documento: etiquetas, atributos, texto, etc.
  • Permite acceder y modificar los elementos HTML desde lenguajes de programación como JavaScript.
  • Es independiente del lenguaje de programación, aunque comúnmente se usa con JS.

Objetivos y utilidad en el desarrollo web

El DOM facilita la interacción dinámica con las páginas web, permitiendo modificar su contenido sin recargarla. Se usa para validar formularios, crear efectos interactivos, modificar estilos en tiempo real y manipular eventos del usuario.
